# 📚 BiblioTech — Application de Gestion de Bibliothèque (Java)

Projet académique réalisé dans le cadre d’un travail de groupe à **l’ECE Paris**, dans le but de concevoir une application Java complète permettant la gestion d’une bibliothèque : livres, membres, emprunts, retours et recommandations.

Ce projet a été développé en collaboration avec deux camarades, dans un contexte pédagogique visant à mettre en pratique la programmation orientée objet, la conception logicielle et la gestion de données.

---

## 🎯 Objectifs du projet

- Concevoir une application Java intuitive et fonctionnelle.
- Gérer l’ensemble des opérations d’une bibliothèque (livres, membres, emprunts).
- Mettre en place une base de données relationnelle.
- Développer une interface utilisateur avec **Swing** ou **JavaFX**.
- Implémenter des fonctionnalités avancées : recherche, historique, recommandations.

---

## 🧠 Fonctionnalités principales

### 📚 Catalogue de livres
- Ajout, modification et suppression de livres.
- Recherche par titre, auteur, genre, disponibilité, etc.

### 👥 Gestion des membres
- Inscription et gestion des informations.
- Suivi de l’historique des emprunts.

### 🔄 Emprunts et retours
- Enregistrement des prêts.
- Gestion des retards et pénalités.
- Mise à jour automatique de la disponibilité des ouvrages.

### ⭐ Recommandations
- Suggestions basées sur les préférences et l’historique de lecture.

---

## 🏗️ Architecture & Technologies

- **Java (OOP)**
- **Swing / JavaFX** pour l’interface graphique
- **MySQL** pour la base de données
- **JDBC** pour la communication avec la base
- Architecture modulaire (classes dédiées aux entités, services, gestion des données)
